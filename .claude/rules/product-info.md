# Sermonize — Product Info

- **Product:** Talk to any sermon — paste a YouTube URL and chat about the details (quotes, verses, themes, timestamps)
- **MVP:** YouTube URL input + chat interface powered by transcript context
- **Tech approach:** Full transcript in LLM context window (no RAG needed — 60-min sermon is ~12K tokens)
- **Local dev:** App runs on `http://localhost:4004`
